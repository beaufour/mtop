Introduction
------------

`mtop` is a `top` like tool for [MongoDB](http://www.mongodb.org/).


Usage
-----

    > mtop

    > mtop -s mongo.superduper.com -d 1500


Dependencies
------------

* [python 2.5+](http://python.org/)
* [pymongo](http://api.mongodb.org/python/)
